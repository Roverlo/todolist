## 问题判断
- 工具栏的通用规则 `.toolbar-actions button { background: #fff; }` 覆盖了 `.primary-btn` 与 `.accent-btn` 的背景色，导致按钮文本颜色仍为 `#fff`，在白底上不可见。
- 代码中按钮文本存在，但由于背景被覆盖、文字为白色所以看起来“空白”。

## 修改方案
- 在 `App.css` 提高两个按钮的样式优先级，显式指定背景与文字颜色：
  - `.toolbar-actions .primary-btn { background: var(--accent) !important; color: #fff !important; }`
  - `.toolbar-actions .accent-btn { background: var(--warning) !important; color: #fff !important; }`
- 保持其他按钮（导入/导出）的白底样式不变。

## 验证
- 工具栏左侧两个按钮显示为绿色与橙色，文字清晰可见；“导入/导出”仍为白底黑字。
- 页面其它地方使用 `.primary-btn` 不受影响。

## 交付
- 仅修改 `app/src/App.css`，不改动 TSX 结构。
