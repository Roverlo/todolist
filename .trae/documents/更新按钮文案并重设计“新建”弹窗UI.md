## 改动目标
- 工具栏：保留两个按钮并更新文案：
  - 绿色按钮：新建单次任务
  - 橙色按钮：新建周期任务
- 点击后分别打开对应类型的弹窗；不再出现“分页/模板列表”等统一视图。

## 实现方案
- 新增两个组件：
  1) `SingleTaskModal.tsx`（单次任务）：
     - 字段与布局按你截图：项目、标题（占位“请输入标题（最多50字符）”）、详情、状态/优先级（并排）、截止日期/现场责任人（并排）、产线责任人、下一步。
     - 逻辑：校验标题≤50；调用 `addTask`；成功后关闭并提示。
  2) `RecurringTaskModal.tsx`（周期任务）：
     - 字段：名称、项目、频率（每周/每月）、周几多选/几号、是否不固定（每周期一条）、到期策略（当天/周末/月底/不设置）、默认字段（现场/产线责任人、下一步）。
     - 逻辑：
       - 保存周期规则：调用 `addRecurringTemplate` 或 `updateRecurringTemplate`（内部状态无 ID 时新增，有 ID 时更新）。
       - 生成本期任务：调用 `materializeRecurringTasks`；成功后关闭并提示。
     - 不包含模板列表页，不做管理视图，只做“新增/保存”和“生成”。
